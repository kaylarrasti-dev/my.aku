Di bawah ini terdapat daftar 20 alat yang umum digunakan dalam pendidikan dan praktik geomatika, beserta deskripsi, fungsi, langkah penggunaan singkat, dan tips keselamatan/kalibrasi.

1. Handy Talky (Motorola)

Deskripsi: Radio komunikasi portabel untuk koordinasi tim lapangan selama survei.

Fungsi utama: Menyediakan komunikasi real-time antar anggota tim, mengurangi kebutuhan kembali ke base.

Langkah penggunaan

  1. Periksa baterai dan frekuensi yang digunakan (pastikan legal).
  2. Nyalakan dan set kanal yang sama untuk semua unit.
  3. Gunakan push-to-talk saat berbicara, lepaskan untuk mendengar.

Tips: Simpan unit pada kondisi kering, gunakan earmic jika berada di area bising.

2. GPS Handheld — Garmin GPSmap 62S

Deskripsi: Penerima GNSS portabel untuk penentuan posisi titik lapangan.

Fungsi: Menentukan koordinat WGS84/UTM untuk dokumentasi titik lapangan.

Langkah penggunaan

  1. Nyalakan unit dan biarkan mencari satelit (ideal 2–5 menit).
  2. Set proyeksi/ datum sesuai kebutuhan proyek.
  3. Kumpulkan beberapa pengukuran di titik yang sama untuk stabilitas.

Tips: Hindari posisi di bawah pohon lebat atau di antara bangunan tinggi.

3. GPS Handheld — Garmin GPSmap 64S

Deskripsi: Versi handheld GPS lain yang umum dipakai untuk survey lapangan sederhana.

Fungsi: Pengumpulan titik posisi, waypoint, dan navigasi lapangan.

Langkah penggunaan

  1. Atur format koordinat, simpan waypoint dengan nama jelas.
  2. Ekspor data ke perangkat lunak SIG untuk pemrosesan.

Tips: Gunakan mode averaging jika tersedia untuk meningkatkan akurasi.

4. GNSS Receiver — Trimble R8

Deskripsi: Receiver GNSS kelas surveyor untuk pengukuran RTK/PPK dengan akurasi tinggi.

Fungsi: Menyediakan solusi posisi sentimeter melalui RTK/PPK.

Langkah penggunaan

  1. Pasang antena pada tiang, sambungkan ke receiver.
  2. Konfigurasi base/rover, set komunikasi modem atau NTRIP.
  3. Lakukan inisialisasi hingga posisi converge.

Tips: Lakukan kalibrasi antena dan periksa log kualitas satelit (SNR).

5. GNSS Receiver — Leica GS18

Deskripsi: Receiver GNSS modern dengan kemampuan RTK dan sensor inersia untuk tilt compensation.

Fungsi: Mengukur titik tanpa harus memastikan vertikal tiang secara sempurna (tilt-enabled).

Langkah penggunaan

  1. Kalibrasi IMU bila perlu, pilih mode tilt jika diinginkan.
  2. Hubungkan ke base atau NTRIP untuk RTK real-time.

Tips: Pelajari koreksi tilt dan kompensasi tinggi antena untuk akurasi terbaik.

6. Drone Pemetaan — DJI Phantom 4 RTK

Deskripsi: Drone kecil dengan modul RTK untuk meningkatkan akurasi georeferensi foto udara.

Fungsi: Akuisisi citra udara untuk fotogrametri dan pembuatan orthomosaic/model 3D.

Langkah penggunaan

  1. Rencanakan misi (overlap, altitude) di software misi.
  2. Periksa cuaca, baterai, dan izin terbang.
  3. Lakukan penerbangan otomatis dan unduh foto RAW untuk pemrosesan.

Tips: Gunakan GCP/RTK untuk hasil georeferensi yang lebih akurat.

7. Drone Industri — DJI Matrice

Deskripsi: Platform drone kelas industri untuk sensor multispectral, LiDAR, atau kamera besar.

Fungsi: Akuisisi data sensor berat untuk misi profesional seperti LiDAR aerial.

Langkah penggunaan

  1. Pasangkan payload (LiDAR/multispectral) dan lakukan pra-check sensor.
  2. Lakukan uji hover sebelum misi penuh.

Tips: Pastikan payload terpasang aman dan baterai cukup untuk misi.

8. LiDAR Scanner — RIEGL

Deskripsi: Sensor LiDAR untuk pemetaan 3D dengan kepadatan titik tinggi.

Fungsi: Menghasilkan point cloud presisi untuk pemodelan permukaan, infrastruktur, dan vegetasi.

Langkah penggunaan

  1. Kalibrasi sensor dan pastikan parameter scannning benar.
  2. Lakukan scanning dari beberapa posisi untuk cakupan penuh.

Tips: Periksa noise dan gunakan filter post-processing untuk membersihkan point cloud.

9. Total Station — Spectra Focus 2"

Deskripsi: Instrumen elektronik untuk mengukur sudut dan jarak (EDM) secara presisi.

Fungsi: Pengukuran topografi, pengaturan titik kontrol, dan staking out pada proyek konstruksi.

Langkah penggunaan

  1. Setup dan level tripod, pasang instrument dan lakukan orientasi/backsight.
  2. Targetkan prisma dan lakukan pengukuran jarak serta sudut.

Tips: Pastikan prisma bersih dan stabil, lakukan pengukuran berulang untuk validasi.

10. Total Station — Geomax Zoom 10

Deskripsi: Total station dengan jangkauan optik yang baik dan fitur otomatis.

Fungsi: Survey topografi dan monitoring perubahan posisi.

Langkah penggunaan

  1. Level dan align instrument ke titik referensi, pilih mode pengukuran.
  2. Simpan data dan ekspor ke perangkat lunak untuk analisis lebih lanjut.

Tips: Simpan konfigurasi proyek untuk mempermudah pengukuran berulang.

11. Total Station — Nikon NPL

Deskripsi: Total station merk Nikon yang biasa dipakai untuk aplikasi survei umum.

Fungsi: Pengukuran sudut dan jarak untuk pembuatan peta dan pekerjaan konstruksi.

Langkah penggunaan

  1. Level instrument, masukkan parameter pengukuran, dan targetkan prisma.
  2. Catat dan verifikasi hasil pengukuran di lapangan.

Tips: Rutin lakukan kalibrasi dan pembersihan lensa optik.

12. Theodolit — Wild T0

Deskripsi: Instrumen optik klasik untuk pengukuran sudut horisontal dan vertikal.

Fungsi: Pengukuran sudut pada survei-tradisional, masih berguna untuk latihan dasar.

Langkah penggunaan

  1. Level dan fokus alat, baca sudut menggunakan vernier/disk digital (jika ada).
  2. Gunakan tripod stabil untuk mengurangi getaran.

Tips: Latih teknik pembacaan sudut untuk akurasi manual yang baik.

13. Theodolit Digital — Sokkia DT 610

Deskripsi: Theodolit dengan pembacaan digital untuk memudahkan pengukuran sudut akurat.

Fungsi: Pengukuran sudut cepat dan akurat, input data digital untuk transfer.

Langkah penggunaan

  1. Inisialisasi alat, set unit sudut (gon/deg).
  2. Ambil beberapa pengukuran untuk validasi.

Tips: Pastikan baterai penuh saat melakukan banyak pengukuran di lapangan.

14. Waterpass — Topcon AT-B 4R

Deskripsi: Instrumen leveling untuk mengukur beda tinggi antar titik.

Fungsi: Pengukuran beda elevasi untuk konstruksi dan survei topografi.

Langkah penggunaan

  1. Levelkan waterpass, baca rambu ukur di titik belakang dan depan.
  2. Hitung selisih elevasi berdasarkan bacaan.

Tips: Gunakan tripod stabil dan lindungi dari cahaya matahari langsung saat membaca.

15. Waterpass — Sokkia B40

Deskripsi: Waterpass portable untuk leveling presisi.

Fungsi: Menentukan beda tinggi yang diperlukan untuk pekerjaan konstruksi dan survei.

Langkah penggunaan

  1. Kalibrasi gelembung dan baca rambu ukur secara konsisten.

Tips: Simpan dalam case pelindung untuk mengurangi risiko kerusakan.

16. Terrestrial Laser Scanner — FARO

Deskripsi: Scanner 3D untuk menangkap detail objek dan situs dalam bentuk point cloud.

Fungsi: Dokumentasi as-built, restorasi bangunan, dan analisis struktur 3D.

Langkah penggunaan

  1. Scan dari beberapa posisi, gunakan target referensi jika perlu untuk registrasi.

Tips: Jaga jarak aman terhadap area yang dipindai dan perhatikan reflektivitas permukaan.

17. RTK Base Station — Trimble

Deskripsi: Base station untuk penyediaan koreksi RTK kepada rover.

Fungsi: Menghasilkan koreksi real-time untuk akurasi sentimeter pada rover GNSS.

Langkah penggunaan

  1. Tempatkan base di titik yang diketahui atau stabil, aktifkan transmisi koreksi.
  2. Pastikan link komunikasi antenna-modem dalam kondisi baik.

Tips: Monitor kualitas koreksi dan uptime base station.

18. GIS Workstation (PC & Software)

Deskripsi: Komputer workstation dengan perangkat lunak SIG untuk pemrosesan data spasial.

Fungsi: Pengolahan data GNSS/Total Station, pembuatan peta, analisis spasial, dan produksi peta akhir.

Langkah penggunaan

  1. Impor data lapangan, lakukan pembersihan data (filter, transformasi).
  2. Lakukan analisis dan ekspor produk peta sesuai kebutuhan.

Tips: Backup proyek, gunakan proyeksi yang konsisten.

19. Software SIG — ArcGIS / QGIS

Deskripsi: Perangkat lunak untuk analisis, visualisasi, dan manajemen data spasial.

Fungsi: Analisis spasial, geoprocessing, pembuatan map layout, dan manajemen database spasial.

Langkah penggunaan

  1. Pelajari alur kerja (import → clean → analyze → layout).
  2. Gunakan plugin dan skrip untuk otomatisasi tugas berulang.

Tips: Dokumentasikan metadata dan gunakan sistem versioning untuk data penting.

20. Total Station — Topcon

Deskripsi: Total station dari Topcon yang banyak dipakai di proyek konstruksi dan survei.

Fungsi: Pengukuran sudut dan jarak presisi dengan kemampuan penyimpanan data dan konektivitas.

Langkah penggunaan

  1. Level dan kalibrasi instrument; set project dan units.
  2. Gunakan prism atau mode reflectorless sesuai kebutuhan pengukuran.

Tips: Periksa firmware dan simpan data proyek secara berkala ke media eksternal.
